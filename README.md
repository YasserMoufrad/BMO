# BMO - AI Student Assistant

BMO, inspired by adventure time, is a personal operating system thats going to start as a smart academic assistant for college students. It will read your syllabus, extract every assignment, due date, and grade weight automatically, and proceed to build a study plan around your life.

## Current Features
- Syllabus Parsing
- Grade weight detection
- Assignment priority detection
- Persistent data storage via JSON

## Built with
- Python
- pdfplumber
- Regular Expressions

## Status
Active deployment -- built from scratch as a learning project.
